# MongoDB-VertexAI-Qwiklab

### Install the pre-requisites
```
sudo apt update
sudo apt install python3-pip
pip3 --version
```
### Run the application
```
cd MongoDB-VertexAI-Qwiklab
streamlit run app.py
```
